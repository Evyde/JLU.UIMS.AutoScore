# JLU_Uims_AutoScore
自动查成绩，好耶！  
## 使用方法
先更改`main.py`中`username`和`password`字段改成自己的`uims`系统中的用户名和密码，然后修改`m.config`这一行，改成您喜欢的推送方式并按照代码传入正确的字典。  
然后：  
`pip install -r requirements.txt`  
`python3 main.py`  
就可以了，等待着~~挂科~~满绩的消息吧~
